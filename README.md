This repo serves two goals:

1. **Teaching** — Complete lesson materials for university ML courses,
   including lesson plans, interactive activities, and student handouts.

2. **Portfolio** — Demonstrates practical ML engineering skills 
   for Applied Scientist and ML Engineer roles.
